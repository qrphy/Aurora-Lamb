# Aurora-Lamb

Aurora-Lamb is a front-end example project that explores a fictional physical-product brand. It’s intended as a compact demo of component-driven UI design, responsive layouts, accessibility considerations, and theming using a TypeScript-first codebase.

## What you’ll find here
- Example product and marketing pages
- Reusable UI components and layout patterns
- Theming and responsive design techniques
- Emphasis on TypeScript for safety and clarity

## Tech stack
- Primary language: TypeScript (≈92.5%)
- Supporting languages: JavaScript, CSS

## Who it’s for
- Front-end developers learning component architecture and TypeScript
- Designers and devs prototyping a small storefront or brand site
- Educators teaching modern front-end patterns
